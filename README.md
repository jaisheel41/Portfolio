# Portfolio Website

A modern, responsive portfolio website showcasing my professional experience, projects, skills, and achievements.

## What's Included

- **About**: Personal introduction and professional background
- **Education**: Academic qualifications and achievements
- **Experience**: Work history and professional roles
- **Skills**: Technical skills and technologies
- **Certifications**: Professional certifications and credentials
- **Projects**: Portfolio of completed projects
- **Blog**: Articles and video content
- **Contact**: Get in touch form

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Django REST Framework
- **Email**: EmailJS for contact form
